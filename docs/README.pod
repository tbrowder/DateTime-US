=begin pod

=head1 NAME

DateTime::US - Provides timezone and DST infomation for US states and territories

=head1 SYNOPSIS

=begin code :lang<raku>

use DateTime::US;

=end code

=head1 DESCRIPTION

Module B<DateTime::US> provides a class used to help Raku programs needing US timezone and DST information.
The use creates a DateTime::US object by providing a US timezone abbreviation.  The list of known data may be
listed without an object:

=begin code
$ raku
> use DateTime::US;
> DateTime::US.show;


=end code
